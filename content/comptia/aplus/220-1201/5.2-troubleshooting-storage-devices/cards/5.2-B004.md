## Question

A workstation displays a "Boot Device Not Found" style message. What should a
technician verify before assuming Windows is corrupted?

## Hint

First decide whether the system can see the storage device at all.

## Answer

**Verify whether the drive is detected and check physical indicators, cables,
power, firmware settings, and the exact boot error.**

## Instructor Notes

### Narrowing

This symptom can happen before the operating system is involved. The first split
is drive detection versus bootability: if the system cannot see the device,
Windows repair is premature.
