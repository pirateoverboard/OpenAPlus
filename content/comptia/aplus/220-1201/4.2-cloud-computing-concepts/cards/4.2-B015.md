## Question

A cloud storage service keeps the same current files available across multiple locations and client devices. Which cloud characteristic is this?

## Answer

File synchronization.

## Instructor Notes

File synchronization keeps data consistent across places. It is not the same as elasticity, which changes resource capacity.
