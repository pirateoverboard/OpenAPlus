## Question

A cloud service uses redundant systems in multiple locations so users can still access the service if one location has a problem. Which cloud characteristic is this?

## Answer

Availability.

## Instructor Notes

Availability focuses on keeping the service reachable. Redundancy can support availability, but redundancy itself is not the characteristic named in the objective.
