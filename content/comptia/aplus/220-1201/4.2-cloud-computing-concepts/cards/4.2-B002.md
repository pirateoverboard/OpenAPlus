## Question

An organization runs a cloud environment restricted to its own users and workloads. Which cloud deployment model is this?

## Answer

Private cloud.

## Instructor Notes

The key clue is dedicated use by one organization. A private cloud can still use cloud-style automation and virtualization; it is not defined only by physical location.
