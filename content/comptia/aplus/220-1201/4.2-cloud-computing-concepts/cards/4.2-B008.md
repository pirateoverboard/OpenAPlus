## Question

Users access a complete cloud-hosted application without installing or maintaining the application locally. Which service model is this?

## Answer

Software as a Service (SaaS).

## Instructor Notes

SaaS is the finished application model. The customer still manages things like user access and data choices, but not the application hosting stack.
