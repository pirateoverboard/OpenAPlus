## Question

A company keeps sensitive internal systems in its private cloud but runs customer-facing web services in a public cloud. Which deployment model combines these environments?

## Answer

Hybrid cloud.

## Instructor Notes

Hybrid cloud combines private and public cloud resources. The card tests the combination, not which workload belongs in each environment.
