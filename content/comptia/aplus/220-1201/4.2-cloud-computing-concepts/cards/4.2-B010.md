## Question

A cloud service tracks CPU time, storage use, and data transfer so customers can be charged based on actual usage. Which cloud characteristic is this?

## Answer

Metered utilization.

## Instructor Notes

Metering measures consumption. It can support billing, reporting, or chargeback, but it is not the same as automatically adding or removing resources.
