## Question

What does CPU core configuration describe?

## Answer

It describes how many processing cores are available in the CPU, such as dual-core, quad-core, octa-core, or multicore.

## Instructor Notes

### Distinction

More cores can improve parallel work when software can use them, but core count is not the same thing as multiple physical CPU sockets.
