## Question

What does a multisocket motherboard support?

## Answer

A multisocket motherboard supports more than one physical CPU package on the same motherboard.

## Instructor Notes

### Distinction

Multisocket is not the same as multicore. Multisocket means multiple CPU packages; multicore means multiple cores inside a CPU package.
