## Question

What does changing the boot order in BIOS/UEFI control?

## Answer

It controls which bootable device the system tries first, and which devices it tries next if the first option does not boot.

## Instructor Notes

### Why

Boot order is a firmware setting. It is commonly changed when installing an operating system, recovering a system, or preventing unauthorized USB boot.
