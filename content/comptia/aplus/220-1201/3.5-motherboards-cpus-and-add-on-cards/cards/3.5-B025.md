## Question

When would a technician install a video card?

## Answer

Install a video card when the system needs dedicated graphics processing or display output beyond what the CPU or motherboard provides.

## Instructor Notes

### Distinction

A video card outputs graphics to displays. A capture card receives video input from an external source.
