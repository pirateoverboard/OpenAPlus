## Question

Why should a technician avoid describing a hard drive or SSD as RAM?

## Answer

RAM is working memory used while software is running; a hard drive or SSD is long-term storage that keeps data after power is removed.

## Instructor Notes

### Common Confusion

Both RAM and storage affect system performance, but they solve different problems. RAM holds active data for the CPU, while drives provide persistent storage.
