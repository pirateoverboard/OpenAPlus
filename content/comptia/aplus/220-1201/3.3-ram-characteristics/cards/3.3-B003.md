## Question

A motherboard supports DDR4 memory. Why is DDR5 not a safe replacement choice even if the capacity looks acceptable?

## Answer

DDR generations are not backward-compatible; the motherboard, slot keying, and supported memory generation must match.

## Instructor Notes

### Why

Capacity is only one RAM characteristic. A replacement module must match the platform's supported DDR generation, not just the desired number of gigabytes.
