## Question

Which RAM type is more appropriate for a critical server that should detect and correct certain memory errors: ECC or non-ECC?

## Answer

**ECC RAM** is more appropriate because it can detect and correct certain memory errors while the system is running.

## Instructor Notes

### Exam Trap

ECC is associated with reliability-focused systems such as servers. Non-ECC memory is common in consumer systems but does not provide the same correction capability.
