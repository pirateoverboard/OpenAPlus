## Question

A system should use dual-channel memory. What installation habit best supports that goal?

## Answer

Install closely matched memory modules in the motherboard slots specified for the channel configuration.

## Instructor Notes

### Why

Channel mode depends on the motherboard and memory controller. Matching modules and using the correct slots give the system the best chance of using the intended channel configuration.
