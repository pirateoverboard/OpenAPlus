## Question

What is the purpose of a multi-channel RAM configuration such as dual-channel memory?

## Answer

It increases CPU-to-RAM throughput by using more than one memory channel.

## Instructor Notes

### Why

The channel count describes memory paths, not the amount of RAM by itself. Two modules installed correctly can provide more memory bandwidth than one comparable module.
