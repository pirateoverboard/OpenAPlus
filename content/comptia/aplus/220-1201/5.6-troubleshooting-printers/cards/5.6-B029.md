## Question

Jobs leave the user's computer but never reach the printer, and the environment
uses a shared print server. What should you check?

## Answer

**Check the print server or print server queue in the path.**

## Instructor Notes

### Why

The print server can be part of the printer path. If jobs leave the client but
do not arrive at the printer, the server or shared queue is a logical boundary
to test.
