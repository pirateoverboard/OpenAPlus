## Question

A phone can make calls, but the user does not want it to use cellular data. What should you check before enabling airplane mode?

## Answer

Check whether **cellular data can be disabled separately** while leaving voice service enabled.

## Instructor Notes

### Why

Airplane mode disables cellular networks broadly. Some devices allow a narrower setting that disables mobile data without fully disabling voice service.

### Exam Trap

Do not use airplane mode as the first answer when the scenario says voice should remain available.
