## Question

A user needs to download a large file on a phone while avoiding cellular data usage. Which network connection should they prefer when available?

## Answer

Use **Wi-Fi** through a local 802.11 access point.

## Instructor Notes

### Why

Wi-Fi can provide Internet access without consuming cellular bandwidth, but it depends on local access-point coverage.

### Real Help Desk

For large downloads or synchronization jobs, verify the device is actually on Wi-Fi before starting the transfer.
