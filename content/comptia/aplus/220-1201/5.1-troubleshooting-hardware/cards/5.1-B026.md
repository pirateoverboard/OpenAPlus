## Question

Only one application closes unexpectedly and the user does not remember the
error text. What OS evidence source can help find crash details?

## Hint

Look for a system-maintained record of application failures.

## Answer

**Check Event Viewer for the application crash entry.**

## Instructor Notes

### Reasoning

Event Viewer can provide a timestamped record when the user cannot repeat the
exact message. That makes the next step more evidence-based than reinstalling
immediately.
