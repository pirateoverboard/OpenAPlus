## Question

A user says their PC tries to boot from a USB installer they left plugged in. What is the best first fix?

## Hint

Rule out simple removable-media causes before changing firmware settings.

## Answer

**Remove the bootable media and confirm the BIOS or UEFI boot order if the problem returns.**

## Instructor Notes

### Help Desk Logic

Start with the simple, reversible cause. If removable media caused the symptom once, boot order should be checked only if the system keeps selecting the wrong startup device.
