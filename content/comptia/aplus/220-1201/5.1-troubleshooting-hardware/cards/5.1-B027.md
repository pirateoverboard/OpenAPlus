## Question

An application has crashed several times this week, but the user cannot give
exact times. What Windows history view can help spot the pattern?

## Answer

**Check Reliability Monitor for the application failure history.**

## Instructor Notes

### Reasoning

Reliability Monitor is useful when the pattern matters more than one exact
message. It can show repeated application failures across days and help decide
whether the issue follows an update, install, or other change.
