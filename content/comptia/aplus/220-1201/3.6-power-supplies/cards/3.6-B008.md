## Question

When is a redundant power supply the appropriate choice?

## Answer

Use redundant power supplies when the system must keep running after one power supply module fails, such as in server hardware.

## Instructor Notes

### Why

Redundancy is about availability, not extra performance. Each supply must be able to support the load so a failed module can be replaced without stopping the supported system.
