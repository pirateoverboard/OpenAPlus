## Question

A user can browse websites and receive email, but cannot reach a shared Windows folder or printer. Which service protocol should be investigated for file and printer sharing?

## Answer

Check **SMB**.

## Instructor Notes

### Why

SMB is the Windows file-sharing and printer-sharing protocol family. This card tests service recognition, not the exact port recall covered by `2.1-C013`.

### Exam Trap

Do not troubleshoot this as a web, mail, or DNS issue when the symptom is specifically Windows file/printer sharing.
