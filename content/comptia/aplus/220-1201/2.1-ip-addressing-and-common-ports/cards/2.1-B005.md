## Question

A technician needs remote terminal access to a network device, but credentials should not be sent in clear text. Which protocol should replace Telnet?

## Answer

Use **SSH**.

## Instructor Notes

### Why

SSH provides an encrypted remote communication link. Telnet provides remote console-style access but sends communication in clear text.

### Exam Trap

Keep the security distinction separate from the port numbers: SSH uses TCP 22, and Telnet uses TCP 23.
