## Question

A user can reach a server by IP address but not by its hostname. Which service path should you check first?

## Answer

Check **DNS** name resolution.

## Instructor Notes

### Why

DNS converts names to IP addresses. If direct IP access works but names fail, name resolution is the focused path to investigate.

### Exam Trap

Do not start by replacing the network adapter when the symptom points to name resolution.
