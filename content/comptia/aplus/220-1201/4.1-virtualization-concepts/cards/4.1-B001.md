## Question

What is a virtual machine?

## Answer

A virtual machine is a software-defined computer that runs as a guest with its own operating system, resources, and settings on shared physical hardware.

## Instructor Notes

### Why

The host owns the real hardware, but each guest behaves like a separate computer. This separation lets one physical system run multiple operating systems or workloads.
