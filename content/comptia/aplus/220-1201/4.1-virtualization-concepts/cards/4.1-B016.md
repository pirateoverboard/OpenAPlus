## Question

Why are containers usually lighter than full virtual machines?

## Answer

Containers share the host operating system kernel instead of running a separate guest operating system for each application instance.

## Instructor Notes

### Distinction

A VM isolates a full operating system. A container isolates an application and its dependencies while relying on the host OS kernel.
