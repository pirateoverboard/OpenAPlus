## Question

A business app only runs correctly on an older operating system. How can virtualization help?

## Answer

Run the older operating system in a VM and install the legacy application inside that guest.

## Instructor Notes

### Why

This keeps the old application environment available without requiring the entire physical computer to run the older operating system.
