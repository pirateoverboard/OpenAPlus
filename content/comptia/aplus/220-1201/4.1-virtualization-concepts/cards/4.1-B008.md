## Question

A technician installs virtualization software as an app inside Windows. Which hypervisor type is this?

## Answer

Type 2, because the hypervisor is hosted by an existing operating system.

## Instructor Notes

### Why

Hosted hypervisors are common for desktop labs and client-side testing. The host OS starts first, and the VMs run above that host OS.
