## Question

Why must a virtualization host be sized for CPU, memory, and storage?

## Answer

The host platform needs suitable CPU virtualization support, enough memory for the host and guests, and enough storage for each guest's virtual disk image.

## Instructor Notes

### Distinction

Virtualization adds overhead. CPU virtualization support belongs to the host platform, while each guest consumes allocated CPU time, memory, storage, and virtual hardware.
