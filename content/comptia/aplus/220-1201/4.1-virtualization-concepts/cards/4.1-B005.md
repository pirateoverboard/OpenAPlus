## Question

What problem does cross-platform virtualization solve?

## Answer

It lets a user run operating systems or applications from different platforms on the same physical computer without rebooting between them.

## Instructor Notes

### Why

No single operating system runs every tool. Cross-platform virtualization lets a user move between guests, such as Windows, Linux, or macOS environments, on one host.
