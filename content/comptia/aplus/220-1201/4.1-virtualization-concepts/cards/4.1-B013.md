## Question

What is Virtual Desktop Infrastructure?

## Answer

VDI provides desktop sessions or applications from remote infrastructure while the client device mainly displays the session and sends user input.

## Instructor Notes

### Why

VDI can reduce endpoint hardware requirements because the desktop workload runs elsewhere. Hosted virtual desktops may also be described as Desktop as a Service.
