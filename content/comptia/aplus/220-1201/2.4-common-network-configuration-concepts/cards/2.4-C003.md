## Text

A DNS {{c1::CNAME}} record makes one name an alias of another host name.

## Extra

The alias follows the canonical name rather than storing a separate IP address.

## Instructor Notes

### Why

CNAME is useful when multiple service names should resolve through one canonical host name.
