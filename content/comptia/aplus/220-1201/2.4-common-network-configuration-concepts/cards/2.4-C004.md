## Text

A DNS {{c1::MX}} record identifies the host name of a domain's mail exchanger.

## Extra

The named mail host requires its own address record for IP resolution.

## Instructor Notes

### Exam Trap

An MX record points to a host name, not directly to the mail server's IP address.
