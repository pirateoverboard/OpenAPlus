## Question

During an HDD-to-SSD migration, the user wants to preserve the existing operating system, applications, settings, and documents. Which migration approach best meets that requirement?

## Answer

**Image or clone the HDD to the SSD** with appropriate imaging software.

## Instructor Notes

### Why

Cloning or restoring an image carries the existing software environment forward, avoiding a complete operating-system and application reinstall.

### Exam Trap

A clean installation is a different valid strategy, but it does not satisfy the requirement to preserve the existing installation directly.
