## Question

What is the practical difference between creating a drive image file and performing a direct drive-to-drive clone?

## Answer

An **image** creates an intermediate file; a **clone** copies directly between drives.

## Instructor Notes

### Why

Both approaches can migrate an existing installation, but they differ in whether an intermediate image file is created.

### Real Help Desk

Either workflow requires suitable imaging software and a way to connect the needed storage devices.
