## Question

A user moved a desktop to a new desk. The computer powers on, but the monitor
shows "No signal." What should you check first before replacing the monitor?

## Hint

Start with the simplest external display cause.

## Answer

**Confirm that the monitor has power.**

## Instructor Notes

### Troubleshooting Logic

This scenario changed the physical display path, but power is still the first
simple external check. A powered-off monitor can make a working computer look
like a display failure.
