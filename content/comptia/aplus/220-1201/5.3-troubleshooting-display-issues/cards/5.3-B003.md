## Question

A PC displays firmware startup screens, but the monitor goes blank when Windows
loads. What startup mode should you try before replacing the monitor?

## Hint

Separate pre-OS video from operating-system display configuration.

## Answer

**Use a low-resolution or recovery display mode.**

## Instructor Notes

### Troubleshooting Logic

The monitor has already shown it can display video before the OS loads. That
points away from immediate monitor replacement and toward a Windows display
configuration or driver path that can be isolated with a basic startup mode.
