## Question

Windows loads, but the display is using an unsupported or incorrect mode. What
configuration area should you check?

## Answer

**Check the operating-system display settings.**

## Instructor Notes

### Troubleshooting Logic

Display settings can create a symptom after the OS loads even when firmware
screens display correctly. This is a configuration check, not proof that the
monitor has failed.
