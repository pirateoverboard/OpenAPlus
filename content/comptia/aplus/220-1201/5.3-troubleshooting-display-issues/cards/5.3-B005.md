## Question

A user's external LCD monitor works, but text looks fuzzy after the resolution
was changed. What is the best first setting to verify?

## Hint

Match the system output to what the fixed-pixel display is designed to show.

## Answer

**Verify the computer is using the monitor's native resolution, or a compatible
same-aspect setting if native resolution is not available.**

## Instructor Notes

### Troubleshooting Logic

LCDs have a fixed pixel grid. A mismatched resolution can make a working
display look poor, so settings should be checked before replacing hardware.
