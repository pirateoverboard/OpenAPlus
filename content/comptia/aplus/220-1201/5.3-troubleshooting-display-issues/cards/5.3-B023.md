## Question

A monitor's colors look unusually warm or dim late in the day, but the same
display looks normal in firmware screens. What OS feature should you check?

## Hint

Look for a software setting that intentionally changes color output.

## Answer

**Check the operating system's night-light or color mode setting.**

## Instructor Notes

### Why

OS color modes can change the display output without a hardware failure. The
firmware-screen comparison helps separate software configuration from panel
damage.
