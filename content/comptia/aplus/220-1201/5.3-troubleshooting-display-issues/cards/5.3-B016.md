## Question

A user sees lines, flashing colors, and blocks before the operating system
loads. What does that test result suggest?

## Hint

Use the timing of the symptom to decide whether an application is likely to be
the cause.

## Answer

**The issue is less likely to be limited to one application or OS display
setting; continue isolating the hardware display path.**

## Instructor Notes

### Troubleshooting Logic

Artifacts that appear before the OS loads point away from an app-only problem.
That does not prove which component failed, but it supports hardware-path
isolation with cable, display, port, adapter, or graphics hardware tests.
