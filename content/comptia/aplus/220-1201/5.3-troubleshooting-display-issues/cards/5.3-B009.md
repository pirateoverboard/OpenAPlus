## Question

A user says the monitor suddenly looks very blue after working normally
yesterday. What monitor setting should you check before replacing the display?

## Hint

Start with the display's own color controls.

## Answer

**Check the monitor's color preset or tint setting.**

## Instructor Notes

### Troubleshooting Logic

An overall color cast can come from the display's own controls. This card tests
that one local setting before broader display-path troubleshooting.
