## Question

An internet service carries broadband data over a cable-provider network using DOCSIS. Which connection type is this?

## Answer

**Cable internet**.

## Instructor Notes

### Exam Trap

DOCSIS identifies cable-data service. Current DOCSIS versions and provider speed tiers are not needed to recognize the connection type.
