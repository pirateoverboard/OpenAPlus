## Question

Several servers need high-bandwidth network access to shared storage that appears as block devices rather than ordinary shared files. Which network type fits this requirement?

## Answer

A **SAN**.

## Instructor Notes

### Exam Trap

A SAN provides block-level storage access. A normal file share provides file-level access and is not automatically a SAN.
