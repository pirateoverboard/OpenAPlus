## Question

Multiple network devices need to send jobs to one shared printer. Which network service provides shared printing?

## Answer

A **print server**.

## Instructor Notes

### Why

A print server provides printing services for network devices. It may be software on a computer or built into the printer.

### Real Help Desk

When every user loses access to the same printer, check the shared print service path before troubleshooting each workstation separately.
