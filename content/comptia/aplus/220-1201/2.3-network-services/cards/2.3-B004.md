## Question

A department needs centralized storage where users can copy, rename, and delete shared documents. Which network service fits this need?

## Answer

A **file share**.

## Instructor Notes

### Why

File shares centralize access to files while the user interface hides the underlying protocol details.

### Exam Trap

Do not confuse a file share with a database just because both store information.
