## Question

A small organization wants one appliance that combines firewalling, filtering, malware inspection, VPN endpoint, and other security functions. What class of device fits this description?

## Answer

An **all-in-one security appliance**, such as a UTM-style device.

## Instructor Notes

### Why

All-in-one security appliances consolidate multiple security and network functions into one platform.

### Exam Trap

Do not memorize an exhaustive feature list. Recognize the consolidated security-appliance role.
