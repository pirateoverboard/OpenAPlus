## Question

What is the practical difference between 802.11 infrastructure mode and ad hoc mode for printer sharing?

## Answer

Infrastructure mode uses an access point for network access, while ad hoc mode creates a direct wireless link without an access point.

## Instructor Notes

### Distinction

Bluetooth is also wireless, but it is short-range. For normal network printing, infrastructure Wi-Fi is usually the more scalable wireless model.
