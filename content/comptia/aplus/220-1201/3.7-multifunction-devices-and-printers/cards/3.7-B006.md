## Question

When is USB the appropriate printer connectivity choice?

## Answer

Use USB for a direct local connection between the printer and a computer.

## Instructor Notes

### Distinction

USB is simple for one directly connected computer. If many users need access, a network connection, printer share, or print server is usually the more relevant deployment model.
