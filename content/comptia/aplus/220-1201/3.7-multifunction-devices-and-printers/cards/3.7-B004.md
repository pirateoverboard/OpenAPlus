## Question

Why must the printer driver language match the printer's supported language, such as PCL or PostScript?

## Answer

The printer must be able to interpret the page description language sent by the driver; a mismatch can produce garbled or unexpected output.

## Instructor Notes

### Distinction

This is a configuration match, not a network-cabling problem. If data reaches the printer but the language is wrong, the device may print unusable pages.
