## Question

How should a technician choose between scan to email, scan to SMB, and scan to cloud?

## Answer

Choose the destination that matches the workflow: email sends scans to an inbox, SMB sends scans to a network share, and cloud services send scans to cloud storage.

## Instructor Notes

### Distinction

Large scans can fill mailboxes. SMB is useful for Microsoft-style file shares, while cloud scanning fits workflows that already use cloud storage.
