## Question

Which printer security feature helps prevent confidential jobs from sitting unattended in the output tray?

## Answer

Use secured print or badge release so the job prints only after the user authenticates at the device.

## Instructor Notes

### Why

Secured print commonly uses a passcode at the printer. Badging uses the employee badge at the device. Both delay release until the user is present.
