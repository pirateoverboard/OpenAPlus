## Question

A new printer is installed, but users cannot see options for the device's supported trays or duplex settings. What should be checked first?

## Hint

The operating system exposes printer features through software matched to that model.

## Answer

Check that the correct printer driver is installed for the printer model and operating system.

## Instructor Notes

### Why

The driver tells the operating system which printer features are available. The wrong driver can hide trays, duplex options, or language options even when the printer is physically installed correctly.
