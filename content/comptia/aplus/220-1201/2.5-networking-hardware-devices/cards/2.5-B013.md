## Question

Which component provides a device's interface to a network and contains its physical MAC address?

## Answer

The **network interface card (NIC)**.

## Instructor Notes

### Why

The NIC implements the network connection and carries the hardware address used for local frame delivery.

### Exam Trap

A MAC address is a data-link hardware address; it is not the device's IP address.
