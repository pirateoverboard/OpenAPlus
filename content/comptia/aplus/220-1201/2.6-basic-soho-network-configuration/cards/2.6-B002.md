## Question

An IP address is written as hexadecimal groups separated by colons. Which IP version uses this format?

## Answer

**IPv6**.

## Instructor Notes

### Exam Trap

IPv4 normally appears as four decimal octets separated by periods; IPv6 uses hexadecimal and colons.
