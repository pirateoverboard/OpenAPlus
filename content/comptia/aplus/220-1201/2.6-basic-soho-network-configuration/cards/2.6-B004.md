## Question

A host has the correct IPv4 address but incorrectly decides whether destinations are local or require routing. Which IP setting should be checked first?

## Answer

The **subnet mask**.

## Instructor Notes

### Why

The subnet mask tells the host which addresses belong to its local subnet.

### Exam Trap

The default gateway handles off-subnet forwarding, but the mask determines whether the host considers a destination off-subnet in the first place.
