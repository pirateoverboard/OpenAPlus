## Question

A tablet registers touches slightly away from where the user taps. What
software-based fix should you try if the device supports it?

## Hint

Align the input system with where the user actually touches.

## Answer

**Run touch calibration.**

## Instructor Notes

### Why

Cursor drift is an input-alignment symptom. Calibration is a targeted software
step before assuming the screen assembly is failed.
