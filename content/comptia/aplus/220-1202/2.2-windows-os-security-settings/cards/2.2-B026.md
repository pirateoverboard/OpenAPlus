## Question

A computer object belongs under a different departmental OU. In Active
Directory Users and Computers, what should the administrator do?

## Answer

Move the computer object to the target organizational unit.

## Instructor Notes

Moving the object changes its place in the administrative hierarchy and can
change which OU-scoped policies apply. It does not copy or recreate the
computer account.
