## Question

An organization wants each user's Documents folder to store its contents on a
network share while keeping the familiar Documents path. What should be
configured?

## Answer

Folder redirection through Group Policy.

## Instructor Notes

Folder redirection changes the backing location of a known Windows folder. A
home folder is a separately assigned personal network location.
