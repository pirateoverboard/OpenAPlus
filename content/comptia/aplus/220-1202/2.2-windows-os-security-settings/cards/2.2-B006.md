## Question

A user wants one Windows sign-in connected to Microsoft services with supported
settings synchronized across devices. Should the user choose a local account
or a Microsoft account?

## Answer

A Microsoft account.

## Instructor Notes

A local account belongs to one device. A Microsoft account can integrate cloud
services and synchronize supported settings; neither choice by itself defines
whether the account has administrator rights.
