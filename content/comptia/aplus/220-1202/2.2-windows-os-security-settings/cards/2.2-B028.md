## Question

An administrator must enforce a password security requirement for domain users
from a central policy. Which Active Directory feature should be applied?

## Answer

Group Policy.

## Instructor Notes

Group Policy centrally configures user and computer settings. A log-in script
runs after sign-in and is not the control that enforces an authentication rule.
