## Question

An organization wants Windows users to sign in with facial recognition, a
security key, or another Windows Hello method instead of a password. What
authentication approach is it configuring?

## Answer

Passwordless authentication through Windows Hello.

## Instructor Notes

Passwordless describes the sign-in path, not unrestricted access. Windows
still applies identity and authorization policy, and another factor or recovery
method may exist.
