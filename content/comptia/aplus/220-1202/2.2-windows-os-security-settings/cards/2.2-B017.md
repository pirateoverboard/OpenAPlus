## Question

An administrator wants a parent folder's NTFS permission settings to propagate
to its child folders and files. Which permission behavior should be used?

## Answer

Permission inheritance.

## Instructor Notes

Inherited entries come from the parent. An explicit permission set directly on
a child object takes precedence over an inherited entry.
