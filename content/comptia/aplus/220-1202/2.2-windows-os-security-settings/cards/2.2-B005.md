## Question

Only one approved Windows program should receive a firewall exception. Which
type of firewall rule best matches that requirement?

## Answer

An application rule that permits the approved program.

## Instructor Notes

An application rule follows the program requirement. A broad port exception
could also permit unrelated software using that port.
