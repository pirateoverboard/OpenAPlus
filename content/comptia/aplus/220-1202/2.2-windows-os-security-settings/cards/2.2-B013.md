## Question

A domain user should authenticate once and then reach permitted network
resources without entering credentials separately for each resource. Which
log-in capability provides this experience?

## Answer

Single sign-on (SSO).

## Instructor Notes

SSO reduces repeated authentication prompts. It does not grant access beyond
the permissions assigned to the user's identity.
