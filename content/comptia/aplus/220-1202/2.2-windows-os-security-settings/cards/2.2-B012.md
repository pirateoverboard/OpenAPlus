## Question

A Windows sign-in must verify an inherent physical characteristic rather than
something the user types. Which two biometric sign-in options listed in
Objective 2.2 meet that requirement?

## Answer

Fingerprint recognition or facial recognition.

## Instructor Notes

Both are biometric options. A PIN is something the user knows, even when it is
used through Windows Hello.
