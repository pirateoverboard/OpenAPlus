## Question

An organization must encrypt an entire removable USB flash drive using a
Windows volume-encryption feature. Which feature should it use?

## Answer

BitLocker To Go.

## Instructor Notes

BitLocker To Go is the removable-volume form of BitLocker. EFS instead protects
selected files or folders on NTFS.
