## Question

A new Windows computer must use centrally managed domain authentication and
receive domain management. What Active Directory action is required?

## Answer

Join the computer to the Active Directory domain.

## Instructor Notes

Joining establishes the computer as a domain participant. A workgroup device
remains independently managed even if it can reach resources on the network.
