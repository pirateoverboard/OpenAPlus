## Question

Each domain user needs an assigned personal network location that administrators
can manage and back up centrally. What should be assigned to the user profile?

## Answer

A network home folder.

## Instructor Notes

A home folder is the user's assigned network storage location. Folder
redirection instead changes the storage location of known Windows folders such
as Documents or Desktop.
