## Question

Which Windows security feature presents a consent or credential boundary before
software receives administrative access?

## Answer

User Account Control (UAC).

## Instructor Notes

UAC limits silent elevation. Its secure desktop also makes it harder for
another process to automate interaction with the elevation prompt.
