## Question

An administrator wants a defined sequence of tasks to run automatically when
members of a department sign in. Which Active Directory setting should be
assigned?

## Answer

A log-in script assigned through Group Policy to the appropriate users, group,
or organizational unit.

## Instructor Notes

The script performs sign-in automation. Group Policy provides the assignment
mechanism and scope.
