## Question

A technician is signed in to Windows and needs one management application to
edit protected system files. How should the application be launched?

## Answer

Use **Run as administrator** to launch that application with elevated rights.

## Instructor Notes

Membership in Administrators does not make every process elevated by default.
Elevating the required program preserves standard privilege for routine work.
