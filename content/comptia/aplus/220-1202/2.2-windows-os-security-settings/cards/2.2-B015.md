## Question

Which Windows permission layer applies to a file both when it is opened locally
and when it is reached through a network share?

## Answer

NTFS permissions.

## Instructor Notes

Share permissions apply only across the network. A network request can be
limited by both share and NTFS permissions, while local access is not governed
by share permissions.
