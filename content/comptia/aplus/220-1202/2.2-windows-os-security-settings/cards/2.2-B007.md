## Question

An employee needs to use ordinary applications but should not make protected
operating-system changes. Which Windows account type should be assigned?

## Answer

A standard user account.

## Instructor Notes

Routine work should use standard privilege. A specific administrative task can
be elevated separately without giving every process unrestricted rights.
