## Question

A user wants a short sign-in code associated with the local Windows device
instead of typing the account password. Which log-in option fits?

## Answer

A personal identification number (PIN).

## Instructor Notes

The device association distinguishes this PIN use from entering the account's
reusable username-and-password credentials.
