## Question

A Windows host must accept inbound traffic identified by one required TCP
port. Which firewall control should be configured?

## Answer

Create an inbound port rule for the required TCP port.

## Instructor Notes

A port rule scopes the exception to the stated protocol and port. Opening more
ports or disabling the firewall would expose unrelated services.
