## Question

On a current Windows system, should the legacy Power Users group be treated as
a reliable privilege level between standard user and administrator?

## Answer

No. Its former elevated permissions were removed, so it is effectively similar
to a standard user on current Windows versions.

## Instructor Notes

The group name is a historical trap. Do not select it when a task actually
requires administrative elevation.
