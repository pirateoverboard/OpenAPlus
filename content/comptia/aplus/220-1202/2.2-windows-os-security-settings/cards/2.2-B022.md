## Question

A user needs to encrypt only selected files and folders on an NTFS volume,
without encrypting the whole volume. Which Windows feature fits?

## Answer

Encrypting File System (EFS).

## Instructor Notes

EFS works at the file-system object level. BitLocker is the choice when the
entire volume must be encrypted.
