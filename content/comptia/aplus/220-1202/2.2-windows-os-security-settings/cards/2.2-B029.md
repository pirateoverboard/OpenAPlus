## Question

Many users need the same access to a network resource, and membership will
change over time. What is the efficient Active Directory permission approach?

## Answer

Assign the resource permissions to a security group, then add or remove users
from that group.

## Instructor Notes

Group-based assignment avoids repeatedly editing the resource permissions for
individual users. OU placement organizes and scopes policy; it is not a
substitute for the security group permission assignment.
