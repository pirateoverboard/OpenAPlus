## Question

A technician knows a filename and needs to locate that file in a directory
tree. Should the technician start with `grep` or `find`?

## Answer

Use `find`.

## Instructor Notes

`find` locates filesystem entries that match criteria such as a name. `grep`
searches text for a matching pattern.
