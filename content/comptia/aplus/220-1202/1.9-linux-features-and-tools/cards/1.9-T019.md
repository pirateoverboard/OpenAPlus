## Prompt

Type the Linux command named by Objective 1.9 that queries DNS information.

## Typed Answer

dig

## Back

`dig` requests detailed information from DNS servers.

## Instructor Notes

Use `ping` for reachability and `traceroute` for the path. `dig` is the
DNS-specific tool among those commands.
