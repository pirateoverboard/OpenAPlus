## Question

A technician needs to review configured DNS resolver details, not a local
hostname override. Which file should be checked?

## Answer

`/etc/resolv.conf`.

## Instructor Notes

`/etc/hosts` stores local name-to-address mappings. `/etc/resolv.conf` holds
resolver configuration used for DNS.
