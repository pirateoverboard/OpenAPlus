## Prompt

Type the base Linux command that changes read, write, or execute permissions.

## Typed Answer

chmod

## Back

`chmod` changes an object's access mode.

## Instructor Notes

Use `chown` for ownership. Permission modes can be expressed symbolically or
with octal notation.
