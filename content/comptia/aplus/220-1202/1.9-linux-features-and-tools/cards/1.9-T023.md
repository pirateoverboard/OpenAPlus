## Prompt

Type the Linux command that provides a continuously updated process and
resource-utilization view.

## Typed Answer

top

## Back

`top` displays live process, CPU, memory, and load information.

## Instructor Notes

`ps` produces a process listing. `top` is the better match when the prompt
emphasizes a live resource view.
