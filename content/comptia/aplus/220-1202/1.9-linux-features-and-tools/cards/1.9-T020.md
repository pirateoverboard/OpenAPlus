## Prompt

Type the Linux command that maps the network path toward a destination.

## Typed Answer

traceroute

## Back

`traceroute` discovers successive hops toward a destination.

## Instructor Notes

Some devices do not return the ICMP responses used for path discovery, and
firewalls can filter them.
