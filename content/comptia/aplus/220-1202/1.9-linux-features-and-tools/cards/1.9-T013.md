## Prompt

Type the Linux command placed before another command to run that command with
superuser or another user's authority.

## Typed Answer

sudo

## Back

`sudo` elevates the specified command.

## Instructor Notes

This narrower scope differs from `su`, which changes the shell's user identity
until the session is exited.
