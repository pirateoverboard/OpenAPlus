## Question

A file has the correct owner, but that owner needs execute permission. Should
the technician use `chmod` or `chown`?

## Answer

Use `chmod` to change the file's access mode.

## Instructor Notes

`chmod` controls read, write, and execute permissions. `chown` changes
ownership, so it would not address the stated problem.
