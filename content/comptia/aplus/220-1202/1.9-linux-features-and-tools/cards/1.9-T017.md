## Prompt

Type the base Linux command that tests IP reachability with ICMP.

## Typed Answer

ping

## Back

`ping` tests reachability and reports round-trip timing.

## Instructor Notes

Use `dig` for DNS information and `traceroute` for path discovery; `ping` is
the reachability tool among these commands.
