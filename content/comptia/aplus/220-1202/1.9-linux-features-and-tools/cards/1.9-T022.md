## Prompt

Type the base Linux command that displays or concatenates file contents.

## Typed Answer

cat

## Back

`cat` sends the contents of one or more files to its output.

## Instructor Notes

`cat` is useful for quick viewing and combination. Use a text editor such as
`nano` when the task is to modify content interactively.
