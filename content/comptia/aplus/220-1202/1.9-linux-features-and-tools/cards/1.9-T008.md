## Prompt

Type the base Linux command that searches text for a pattern.

## Typed Answer

grep

## Back

`grep` finds matching text in input or files.

## Instructor Notes

Use `find` to locate filesystem entries by criteria such as filename; use
`grep` when the target is text content.
