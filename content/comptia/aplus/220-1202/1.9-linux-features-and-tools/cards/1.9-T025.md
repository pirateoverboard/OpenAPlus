## Prompt

Type the base Linux command that reports storage space used by a file or
directory.

## Typed Answer

du

## Back

`du` reports disk usage for specified items.

## Instructor Notes

`df` answers how much free space a filesystem has; `du` helps attribute used
space to files and directories.
