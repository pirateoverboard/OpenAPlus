## Prompt

Type the Linux command that changes the current shell to another user until
that session is exited.

## Typed Answer

su

## Back

`su` switches the shell's user identity.

## Instructor Notes

`sudo` instead runs a specified command with another user's authority without
requiring the shell to remain in that identity.
