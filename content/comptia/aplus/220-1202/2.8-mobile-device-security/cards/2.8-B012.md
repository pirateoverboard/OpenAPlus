## Question

A user must be able to restore mobile data to a replacement phone after the
original is lost or damaged. Which mobile security method provides that
recovery capability?

## Answer

A remote backup application.

## Instructor Notes

Backup preserves a separate recoverable copy. Encryption protects local data,
and remote wipe removes it, but neither control creates the copy needed for
restoration.
