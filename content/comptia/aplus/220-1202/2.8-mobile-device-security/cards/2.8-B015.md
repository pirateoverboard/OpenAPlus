## Question

An organization requires every managed phone to use encryption and an approved
screen lock. Which policy element defines these mandatory controls?

## Answer

Profile security requirements.

## Instructor Notes

These requirements state the approved security controls. A configuration
profile can package the settings so the MDM system can apply them consistently.
