# Changelog

## 2026-07-12 - Independent-review fixes

- Added `1.2-B017` for solid-state/flash and external/hot-swappable boot media.
- Added `1.2-B018` for Internet-based installation sources.
- Added `1.2-B019` for remote network installation as an installation type
  distinct from PXE network boot.
- Reworded `1.2-B009` to test the sourced repair-installation purpose directly
  instead of framing it as an upgrade-choice card.
- Removed duplicate `1.2-C005`; `1.2-B012` remains the MBR limitation card.
- Updated `checklist.md` coverage rows for the added boot/install cards and the
  removed duplicate Cloze card.

## 2026-07-12 - Objective 1.2 draft authoring

- Added the Core 2 Objective 1.2 study guide for boot methods, installation
  types, partitioning, drive formatting, upgrade preparation, feature updates,
  and product lifecycle.
- Added 19 Basic cards and 7 Cloze cards.
- Used CompTIA A+ 220-1202 Exam Objectives v4.0 Objective 1.2 as the primary
  scope authority.
- Used Professor Messer 220-1202 v1.40 pages 3-4 as the approved secondary
  validation/page-reference source.
- Completed a private Professor Messer Core 2 practice-exam gap check; it
  reinforced GPT/MBR selection, third-party driver loading, installation method
  selection, multiboot, zero-touch deployment, and data-preservation coverage
  without being cited in card or study-guide source fields.
- Image cards were considered and omitted because this objective does not need
  visual recognition.

### Scope and source decisions

- Solid-state, external, and Internet-based boot sources remain study-guide
  context because the active selection targets are covered by USB, PXE,
  multiboot, image deployment, and remote/zero-touch installation decisions.
- Exact Windows support durations were not carded because they vary by version
  and edition; lifecycle planning is the stable learning target.
- No new secondary sources were used.
