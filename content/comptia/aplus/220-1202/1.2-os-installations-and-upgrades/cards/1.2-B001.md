## Question

What two conditions must be true to install an OS from a USB drive?

## Answer

The USB drive must be bootable, and the computer must support booting from USB.

## Instructor Notes

A USB installer is local boot media. If either the media is not bootable or the
system cannot boot from USB, the installation cannot start that way.
