## Question

A company ships laptops directly to remote users and wants the installation and
company-specific configuration to complete with minimal user input. Which
deployment approach fits?

## Answer

Zero-touch deployment.

## Instructor Notes

Zero-touch deployment automates the installation and configuration workflow so a
remote user can receive a system that sets itself up.
