## Question

Which installation method deploys a prepared clone or standardized build to
systems quickly?

## Answer

Image deployment.

## Instructor Notes

Image deployment is useful when many devices should receive the same baseline
configuration. It is not the same as preserving each user's existing local
applications and files.
