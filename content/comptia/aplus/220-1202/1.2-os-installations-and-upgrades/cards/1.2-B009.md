## Question

What is a repair installation used for during Windows installation work?

## Answer

It is used to fix problems with the existing Windows OS without modifying user
files.

## Instructor Notes

A repair installation targets the current Windows installation. It is different
from a clean install, which starts over, and from an in-place upgrade, which
moves an existing OS to a newer supported version.
