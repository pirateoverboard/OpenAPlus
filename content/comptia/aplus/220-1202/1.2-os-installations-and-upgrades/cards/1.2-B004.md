## Question

Which installation type should be chosen when the goal is to start over with a
fresh OS environment and the existing data does not need to remain in place?

## Answer

A clean install.

## Instructor Notes

A clean install wipes or replaces the existing environment. Backups and
application reinstall plans matter before choosing it on a user system.
