## Question

Which installation type uses a local server, shared drive, or Internet source
to install the operating system across the network?

## Answer

Remote network installation.

## Instructor Notes

PXE describes booting from the network. Remote network installation describes
where the installation source is located and how the OS installation is
delivered.
