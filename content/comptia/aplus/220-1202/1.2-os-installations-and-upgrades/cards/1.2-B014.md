## Question

Before repartitioning or performing a clean install on a user's drive, what
data-protection step should be completed?

## Answer

Back up existing data and user preferences that may be needed later.

## Instructor Notes

Partitioning, formatting, and clean installs can cause serious data loss.
Protect user data before making destructive storage changes.
