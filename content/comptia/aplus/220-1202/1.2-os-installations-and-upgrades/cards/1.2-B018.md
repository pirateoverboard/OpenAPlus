## Question

A technician starts an OS installation that downloads the installer from an
online source instead of using local USB media or a local network server. Which
boot method category fits?

## Answer

Internet-based installation.

## Instructor Notes

Internet-based installation uses an online source for the installation workflow.
Messer lists examples such as Linux distribution installers, macOS Recovery
installation, and Windows updates.
