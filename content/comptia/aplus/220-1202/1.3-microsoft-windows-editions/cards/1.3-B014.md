## Question

How does Windows 11 Home's Device Encryption differ from BitLocker availability
in Windows 11 Pro?

## Answer

Windows 11 Home includes consumer-oriented Device Encryption, while Windows 11
Pro provides BitLocker full-disk encryption.

## Instructor Notes

This card is about edition comparison. The official objective names BitLocker,
and Messer distinguishes Windows 11 Home Device Encryption from Pro's
BitLocker feature.
