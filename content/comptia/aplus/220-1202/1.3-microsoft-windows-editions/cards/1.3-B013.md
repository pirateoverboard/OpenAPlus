## Question

Why is a 32-bit-only CPU a blocker for a Windows 11 deployment?

## Answer

Windows 11 does not support 32-bit CPUs.

## Instructor Notes

This is a hardware-compatibility distinction between older systems and Windows
11 deployment planning.
