## Question

When is a clean install the better upgrade path than an in-place upgrade?

## Answer

Use a clean install when the goal is to start fresh instead of preserving the
existing applications, files, and settings.

## Instructor Notes

This is the opposite side of the in-place upgrade decision. Clean installs
require data backup and application reinstall planning because the old
environment is not preserved.
