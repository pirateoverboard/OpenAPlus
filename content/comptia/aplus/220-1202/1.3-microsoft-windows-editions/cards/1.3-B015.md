## Question

What should a technician remember about RAM support when comparing Windows
editions?

## Answer

RAM support varies by edition; more advanced editions can support more RAM.

## Instructor Notes

For example, Messer identifies Windows 10 Pro for Workstations as a
high-maximum-RAM edition. Avoid assuming every edition supports the same memory
ceiling.
