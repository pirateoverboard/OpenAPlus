## Question

What is the key difference between a Windows workgroup and a Windows domain?

## Answer

A workgroup is a peer-style set of standalone computers, while a domain uses
centralized authentication and device access for a business network.

## Instructor Notes

The exam distinction is management scope. Workgroups keep control local to each
device; domains centralize identity and access for many devices.
