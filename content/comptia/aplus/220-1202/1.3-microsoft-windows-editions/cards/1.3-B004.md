## Question

A European Windows system is missing Windows Media Player and other multimedia
utilities by design. Which Windows edition variant does this describe?

## Answer

A Windows N edition.

## Instructor Notes

N editions are not a separate management tier. They are Windows editions with
media components removed; the Media Feature Pack can add the missing media
features later.
