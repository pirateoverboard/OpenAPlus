## Question

Which upgrade path should be chosen when the goal is to keep supported
applications, documents, and settings in place?

## Answer

An in-place upgrade.

## Instructor Notes

A clean install starts over and requires backup and application reinstall
planning. In-place upgrade is the preservation-focused path when compatibility
allows it.
