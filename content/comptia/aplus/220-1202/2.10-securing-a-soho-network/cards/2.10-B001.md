## Question

A newly installed SOHO router still uses its factory administrator
credentials. Which security setting should be changed?

## Answer

Change the default administrator password to a strong password.

## Instructor Notes

Default router credentials are easy to obtain, and administrator access allows
configuration changes. This protects the management account; it does not by
itself control which clients may join the network.
