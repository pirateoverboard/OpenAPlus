## Question

Several remote employees need to update the same table of budget values and
formulas. Which cloud collaboration tool best matches the task?

## Answer

A cloud spreadsheet.

## Instructor Notes

### Why

Rows, columns, formulas, and shared numerical data point to a spreadsheet. A
word processor is better suited to a primarily narrative document.
