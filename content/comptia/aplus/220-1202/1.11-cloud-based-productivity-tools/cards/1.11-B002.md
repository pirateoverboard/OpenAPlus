## Question

A user saves a document on a laptop and needs the updated file to become
available automatically on the user's other registered devices. Which cloud
storage feature should be configured?

## Answer

File synchronization.

## Instructor Notes

### Why

Synchronization uploads the local change to cloud storage and makes it
available to the other synchronized devices. Folder settings determine which
content participates and whether it remains stored locally.
