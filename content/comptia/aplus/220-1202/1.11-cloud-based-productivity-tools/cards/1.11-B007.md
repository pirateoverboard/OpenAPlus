## Question

Two employees need to draft and revise the same text-focused policy document
from different locations. Which cloud collaboration tool best matches the
task?

## Answer

A cloud word processing tool.

## Instructor Notes

### Why

The output is a narrative document, which makes word processing the appropriate
category. A spreadsheet is intended for structured rows, columns, and
calculations.
