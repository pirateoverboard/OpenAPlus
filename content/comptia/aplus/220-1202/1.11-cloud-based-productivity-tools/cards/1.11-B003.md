## Question

A laptop has limited local storage. The user needs one project folder available
offline, while older cloud files can remain available on demand. What should
the technician configure?

## Answer

Configure the cloud storage folder settings so the project folder is stored
locally and the older content uses streaming or on-demand access.

## Instructor Notes

### Why

Folder settings control which content consumes local space. The offline
requirement identifies what must remain local; the remaining content can stay
cloud-backed until accessed.
