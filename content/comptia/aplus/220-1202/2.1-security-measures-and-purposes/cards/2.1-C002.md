## Text

An authentication code intended to be valid for only one use is a
{{c1::one-time password or passcode (OTP)}}.

## Extra

Hardware and software tokens can generate OTPs. TOTP is a more specific form
whose changing value is tied to time.

## Instructor Notes

“One-time” describes reuse, while “time-based” describes how a TOTP changes and
expires. Keep the category and its time-based subtype distinct.
