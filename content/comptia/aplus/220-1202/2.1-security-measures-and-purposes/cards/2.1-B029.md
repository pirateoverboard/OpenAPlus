## Question

Which access method lets a user authenticate once and then use assigned
resources without a new credential prompt for each resource during the session?

## Answer

Single sign-on (SSO).

## Instructor Notes

SSO reduces repeated authentication prompts. It does not grant every possible
permission; authorization still determines which resources are assigned.
