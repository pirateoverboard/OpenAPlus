## Text

A {{c1::time-based one-time password (TOTP)}} derives a short-lived code from a
shared secret and the current time.

## Extra

TOTP is commonly generated by an authenticator application. Its time basis
distinguishes it from one-time-code methods based on a different changing value.

## Instructor Notes

TOTP is a type of OTP. The important extra property is that time causes the code
to expire rather than the code remaining valid indefinitely.
