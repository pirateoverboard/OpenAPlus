## Question

Which macOS application provides command-line access for scripts, file
management, and system configuration?

## Answer

Terminal.

## Instructor Notes

Terminal exposes a command-line interface to macOS. This objective tests the
tool's purpose; exact shell commands belong to the task being performed and are
not duplicated as Objective 1.8 command cards.
