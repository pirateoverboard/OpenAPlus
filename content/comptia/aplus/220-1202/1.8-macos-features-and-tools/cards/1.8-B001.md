## Question

A technician needs to remove a self-contained macOS application that has no
dedicated uninstaller. What is the normal removal method?

## Answer

Move the application's `.app` bundle from `/Applications` to the Trash.

## Instructor Notes

Many macOS applications are self-contained bundles. Some applications include
a separate uninstall program; use that program when it is provided.
