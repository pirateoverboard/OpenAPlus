## Question

Which macOS tool is the primary file manager for browsing, moving, renaming,
and deleting files?

## Answer

Finder.

## Instructor Notes

Finder also provides access to file servers, remote storage, and related
resources. Spotlight searches for content, while Finder performs the ongoing
file-management work.
