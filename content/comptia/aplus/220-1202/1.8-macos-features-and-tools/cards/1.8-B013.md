## Question

An approved automation utility must control keyboard and mouse input but macOS
is blocking that control. Which permission category should the technician
review?

## Answer

Accessibility.

## Instructor Notes

Accessibility permission can allow broader input or control capabilities than
ordinary application access. This is distinct from Privacy permissions for
sensitive data and devices such as photos, cameras, or microphones.
