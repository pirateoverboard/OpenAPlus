## Question

What is the difference between `/Library` and `~/Library` in macOS?

## Answer

`/Library` contains support resources shared across the computer, while
`~/Library` contains preferences and support data for the current user.

## Instructor Notes

The tilde means the current user's home directory, so `~/Library` resolves to a
path such as `/Users/<username>/Library`. The official objective's
`/Users/Library` wording is treated as the user-specific Library concept, not
as a literal shared path.
