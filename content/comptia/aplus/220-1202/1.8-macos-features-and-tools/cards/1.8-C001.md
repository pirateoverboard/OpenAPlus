## Text

In macOS, a `.dmg` file is a mountable {{c1::disk image}}.

## Extra

Finder can mount it like a drive, and Disk Utility can create and manage disk
images.

## Instructor Notes

Do not confuse `.dmg` with `.pkg`, which runs an installer, or `.app`, which is
an application bundle.
