## Text

In macOS, a `.pkg` file is an {{c1::installer package}}.

## Extra

Opening it runs an installation process that can place application components
on the system.

## Instructor Notes

A `.pkg` runs an installer process. Some applications also include a separate
uninstall program for removal.
