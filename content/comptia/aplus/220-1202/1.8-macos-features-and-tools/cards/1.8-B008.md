## Question

A widespread macOS security issue needs an urgent fix. Which update mechanism
is designed for this purpose?

## Answer

Rapid Security Response (RSR).

## Instructor Notes

RSR targets urgent security issues and is managed with automatic-update
settings. It complements routine operating-system and application patching; it
does not replace the broader update process.
