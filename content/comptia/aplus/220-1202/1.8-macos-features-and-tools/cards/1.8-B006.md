## Question

A macOS user needs to recover an earlier version of a file from a configured
backup destination. Which built-in feature should the technician use?

## Answer

Time Machine.

## Instructor Notes

Time Machine maintains successive backups for restoration and removes the
oldest retained data as the destination fills. Disk Utility manages storage
and disk images; it is not the versioned backup-and-restore feature.
