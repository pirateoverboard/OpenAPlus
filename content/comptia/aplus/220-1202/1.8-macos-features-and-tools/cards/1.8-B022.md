## Question

A technician needs to inspect a Mac's storage device, modify partition details,
or create a disk image. Which built-in utility should be used?

## Answer

Disk Utility.

## Instructor Notes

Disk Utility manages disks, filesystems, partitions, and disk images and can
verify or repair supported filesystems. Time Machine is used for versioned
backup and restoration, so the tools should not be confused.
