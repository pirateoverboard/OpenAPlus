## Question

A technician needs user account information, while another technician needs to
map a network share. Which `net` command role fits the account-information
task?

## Answer

Account information: use `net user`.

## Instructor Notes

This card tests the `net user` versus `net use` distinction. The typed Command
card tests exact command recall.
