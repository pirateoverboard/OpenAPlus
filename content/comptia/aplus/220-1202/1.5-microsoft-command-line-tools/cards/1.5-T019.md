## Prompt

Which command tests reachability and round-trip time with ICMP?

## Typed Answer

ping

## Back

`ping` tests basic reachability and reports round-trip time.

## Instructor Notes

Some networks filter ICMP, so a failed ping is evidence to interpret, not a
complete diagnosis by itself.
