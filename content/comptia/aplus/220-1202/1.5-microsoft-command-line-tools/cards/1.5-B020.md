## Question

A technician wants active connection information from a Windows client, but
does not need DNS name lookups or route tracing. Which network command category
fits best?

## Answer

Network statistics with `netstat`.

## Instructor Notes

`netstat` focuses on local network statistics and active connections. Use
`tracert` or `pathping` for route-path questions, and `nslookup` for DNS.
