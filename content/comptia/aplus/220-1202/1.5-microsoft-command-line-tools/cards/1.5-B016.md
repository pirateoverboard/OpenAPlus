## Question

After malware cleanup, a technician wants to check whether protected Windows
system files still have integrity. What command-line tool category fits this?

## Answer

System File Checker, using `sfc /scannow`.

## Instructor Notes

System File Checker is for protected system-file integrity. It is not for disk
partitioning, network-share mapping, or DNS lookup.
