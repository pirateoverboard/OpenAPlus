## Question

A technician needs to assign a drive letter to a Windows share, while another
technician needs account information. Which `net` command role fits the drive
mapping task?

## Answer

Share mapping: use `net use`.

## Instructor Notes

This card tests the `net use` versus `net user` distinction. The typed Command
card tests exact command recall.
