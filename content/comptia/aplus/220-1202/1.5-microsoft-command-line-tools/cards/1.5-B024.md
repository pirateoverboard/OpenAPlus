## Question

A technician can reach a host, but wants to see the network path by hop. Which
type of command is more appropriate than a basic reachability test?

## Answer

A route-tracing command such as `tracert`.

## Instructor Notes

This distinction matters because `ping` tests basic reachability, while
`tracert` shows the path by hop.
