## Prompt

Which command form displays expanded IP details such as DNS and DHCP server
information?

## Typed Answer

ipconfig /all

## Back

`ipconfig /all` displays additional network adapter configuration details.

## Instructor Notes

This is useful when basic address output is not enough.
