## Prompt

Which command combines route tracing with packet-loss and latency statistics at
each hop?

## Typed Answer

pathping

## Back

`pathping` combines `ping`-style measurements with a route path.

## Instructor Notes

It is useful when `tracert` alone does not show loss or round-trip statistics
along the path.
