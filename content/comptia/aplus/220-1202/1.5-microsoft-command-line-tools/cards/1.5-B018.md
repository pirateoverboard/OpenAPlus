## Question

A Windows client has an IP address, but the technician also needs DNS and DHCP
server details. Why use the expanded `ipconfig` output instead of the basic
output?

## Answer

It includes additional adapter configuration details such as DNS and DHCP server
information.

## Instructor Notes

The typed Command card tests the exact command form. This card tests why the
expanded output is selected during network troubleshooting.
