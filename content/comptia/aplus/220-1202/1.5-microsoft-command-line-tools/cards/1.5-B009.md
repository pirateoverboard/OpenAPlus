## Question

A technician needs a more capable command-line copy tool for a structured file
copy between folders or systems. Why is Robust Copy the better fit than older
copy utilities?

## Answer

It is designed as a robust Windows file-copy tool for more capable copy jobs.

## Instructor Notes

The typed Command card tests the exact command. This card tests when the robust
copy tool is the right file-management choice.
