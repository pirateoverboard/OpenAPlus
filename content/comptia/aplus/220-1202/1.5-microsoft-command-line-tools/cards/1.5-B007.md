## Question

A technician is preparing a partition for reuse and is warned that the action
can remove access to existing data. Which command category fits this task?

## Answer

A format command.

## Instructor Notes

Formatting prepares the target for use with Windows. It is not a general repair
command, and the technician must confirm the target and data requirements first.
