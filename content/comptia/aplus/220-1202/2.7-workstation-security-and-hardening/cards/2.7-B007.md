## Question

Which BIOS/UEFI password type requires authentication before the workstation
can boot?

## Answer

The firmware user password.

## Instructor Notes

A pre-boot password does not encrypt stored data. If protection is required
after someone removes the drive, data-at-rest encryption addresses that risk.
