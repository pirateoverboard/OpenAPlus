## Question

Which BIOS/UEFI password type should be configured to prevent unauthorized
changes to firmware settings?

## Answer

The firmware supervisor or administrator password.

## Instructor Notes

This password protects configuration access. A firmware user password instead
requires authentication before booting the computer.
