## Question

Users reuse passwords because they cannot remember a different strong password
for every account. Which security option best supports unique passwords?

## Answer

A password manager.

## Instructor Notes

A password manager stores many credentials in one protected database, making
unique passwords practical. Its own access credentials and recovery method
must be secured carefully.
