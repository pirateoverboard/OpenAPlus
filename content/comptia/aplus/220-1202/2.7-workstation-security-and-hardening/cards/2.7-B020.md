## Question

A workstation is running a network-accessible service that no approved
application uses. What hardening action reduces the related attack surface?

## Answer

Disable the unused service.

## Instructor Notes

An active service can provide code or an interface for an attacker to target.
Confirm that it is truly unnecessary before disabling it so required functions
and dependencies are not disrupted.
