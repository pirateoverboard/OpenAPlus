## Question

One employee uses the same strong password for several services. Which
password consideration is missing?

## Answer

Password uniqueness.

## Instructor Notes

A password can meet length and complexity rules yet still create risk through
reuse. Unique passwords keep one compromised service from exposing the same
credential elsewhere.
