## Question

A user has finished working on a shared workstation. Should the user lock the
screen or log off?

## Answer

Log off.

## Instructor Notes

A lock is appropriate for a temporary absence because it preserves the user's
session. Logging off ends the finished session so the next user cannot resume
it under the previous identity.
