## Question

Which workstation hardening setting prevents inserted media from automatically
starting a program through its AutoRun instructions?

## Answer

Disable AutoRun.

## Instructor Notes

AutoRun can initiate a program according to media instructions. AutoPlay is a
related but separate feature that presents handling choices to the user.
