## Question

A user needs a different strong password for each website without memorizing
every credential. Which browser-related security tool fits this requirement?

## Answer

A password manager with an encrypted credential vault.

## Instructor Notes

The manager centralizes protected credential storage while supporting unique
passwords across sites. This is more secure than solving the memory problem by
reusing one password.
