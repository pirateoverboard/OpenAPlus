## Question

A browser is several releases behind and is missing available security fixes.
Which configuration action addresses this risk?

## Answer

Update the browser to a current supported release through its trusted update
mechanism.

## Instructor Notes

The updater may be integrated with the operating system or built into the
browser. The durable security decision is to apply browser patches rather than
memorize a vendor-specific menu path.
