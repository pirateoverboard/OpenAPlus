## Question

An organization requires browser name lookups to use encrypted HTTPS transport
instead of ordinary plaintext DNS. Which browser security setting should be
enabled?

## Answer

Enable secure DNS, such as DNS over HTTPS (DoH), using a DNS service that
supports it.

## Instructor Notes

Secure DNS encrypts the lookup path between the client and the compatible DNS
provider. It protects the DNS request from ordinary plaintext observation; it
is separate from the HTTPS connection to the destination website.
