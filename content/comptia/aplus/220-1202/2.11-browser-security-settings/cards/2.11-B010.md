## Question

A browser profile must no longer retain its history, download list, or saved
website credentials. Which browser action addresses those stored categories?

## Answer

Clear the browsing data and select those categories for removal.

## Instructor Notes

Clearing browsing data is broader than clearing only cached website resources.
The selected categories matter because browsers can retain several distinct
types of local information.
