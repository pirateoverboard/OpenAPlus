## Question

A user needs a browser session that will discard its normal local history,
download list, and cached session information when closed. Which mode should
the user select?

## Answer

Private-browsing mode.

## Instructor Notes

This mode addresses what the browser retains from that local session. It does
not replace browser patching, certificate validation, or the need to obtain
software from trusted sources.
