## Question

An organization requires browsers to send web requests through an explicit
proxy. Which connection details must be configured for the browser?

## Answer

Configure the proxy server's address and port.

## Instructor Notes

An explicit proxy requires client configuration so the browser knows where to
send requests. A transparent proxy works without those browser-specific
settings.
