## Question

A technician receives an optical-disc image as one file and the workstation has
no optical drive. How can the technician expose the installation files to the
operating system?

## Answer

Mount the ISO file so it appears as a separate drive.

## Instructor Notes

An ISO preserves the contents of optical media in a file. Mounting uses the
image directly; physical media would instead require a compatible drive or
port.
