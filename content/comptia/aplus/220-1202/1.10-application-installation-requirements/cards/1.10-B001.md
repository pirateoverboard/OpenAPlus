## Question

A required application is 64-bit, but the target computer runs a 32-bit
operating system. Can the application be installed as a compatible solution?

## Answer

No. A 64-bit application requires a 64-bit operating system.

## Instructor Notes

The reverse relationship is different: 64-bit Windows can commonly run 32-bit
applications. Always compare the application's architecture requirement with
the installed OS.
