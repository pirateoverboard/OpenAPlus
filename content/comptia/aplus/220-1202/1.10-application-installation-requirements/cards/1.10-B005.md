## Question

A workstation has exactly the RAM listed for a new application, but the OS and
several required applications will run at the same time. What should the
technician consider before installation?

## Answer

Whether enough RAM remains for the new application after the OS and concurrent
applications use memory.

## Instructor Notes

An application's RAM requirement is above the memory already needed by the OS.
Insufficient available RAM can cause poor performance or prevent the
application from operating.
