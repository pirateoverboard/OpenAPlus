## Question

An organization needs many matching virtual machines with the OS and required
applications already installed. Which distribution method best fits this goal?

## Answer

Image deployment.

## Instructor Notes

A prepared image installs a complete, consistent system build quickly. It must
match the target platform; virtual machines simplify this because their
presented hardware can be identical.
