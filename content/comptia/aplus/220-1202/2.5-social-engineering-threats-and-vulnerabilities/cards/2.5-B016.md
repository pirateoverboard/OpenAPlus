## Question

An application inserts untrusted input directly into a database query, allowing
the input to alter the query's meaning. What attack is this?

## Answer

Structured Query Language (SQL) injection.

## Instructor Notes

The application failed to keep user-supplied data separate from database
instructions. The manipulated query can expose or change database information.
