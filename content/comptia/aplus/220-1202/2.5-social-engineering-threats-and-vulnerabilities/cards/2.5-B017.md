## Question

A website accepts attacker-supplied script that later executes in a visitor's
browser as though it came from the trusted site. What attack is this?

## Answer

Cross-site scripting (XSS).

## Instructor Notes

XSS targets the browser and the user's trust in a site. This differs from SQL
injection, which changes a query sent to a database.
