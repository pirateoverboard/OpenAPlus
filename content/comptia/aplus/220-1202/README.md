# CompTIA A+ Core 2 (220-1202) content scaffold

This directory is the content root for CompTIA A+ Core 2. The scaffold was
created from the approved official objective source:

- CompTIA A+ 220-1202 Exam Objectives v4.0

No objective in this directory is authored, reviewed, generated, or accepted
until its objective-local `study-guide.md`, `checklist.md`, `changelog.md`,
source cards, generated TSV output, and required verification records are added
according to the repository documentation.

## Scaffold notes

- Every objective has a tracked `cards/` directory.
- Software Troubleshooting objectives also have a tracked `interview/`
  directory for optional objective-local spoken troubleshooting practice.
- Do not add cards, study-guide claims, or checklist coverage until the relevant
  official objective and approved secondary sources have been reviewed.
