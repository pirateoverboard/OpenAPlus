# Objective 1.6 completion checklist

**Objective status: ACCEPTED.** Core 2 Objective 1.6 has passed independent
review, bloat review, blocker-fix verification, generated-output verification,
and manual Anki smoke testing.

Official context:

```text
Domain 1.0 Operating Systems, Objective 1.6: Given a scenario, configure
Microsoft Windows settings.
```

Official bullet scope reviewed:

- Control Panel destinations: Internet Options, Devices and Printers, `Program
  and Features`, Network and Sharing Center, System, Windows Defender Firewall,
  Mail, Sound, User Accounts, Device Manager, Indexing Options, and
  Administrative Tools.
- File Explorer Options: view hidden files, hide extensions, General options,
  and View options.
- Power Options: hibernate, power plans, sleep/suspend, standby, lid-close
  behavior, Fast Startup, and USB selective suspend.
- Ease of Access.
- Windows Settings categories: Time and Language, Update and Security,
  Personalization, Apps, Privacy, System, Devices, Network and Internet,
  Gaming, and Accounts.

The official objective PDF uses `Program and Features`; Professor Messer p. 11
uses the Windows applet name `Programs and Features`. The objective scope is
unchanged, and the validated applet name is used in learner-facing content.
