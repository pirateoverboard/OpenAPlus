## Question

A technician must access a folder that File Explorer does not normally
display. Which settings destination contains the option to show hidden files?

## Answer

File Explorer Options, under the View settings.

## Instructor Notes

The View settings control whether hidden content appears. This is separate
from showing file extensions, even though both choices are in File Explorer
Options.
