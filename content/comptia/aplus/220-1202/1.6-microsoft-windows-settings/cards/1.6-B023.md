## Question

A user wants to change the desktop wallpaper, colors, and lock-screen
appearance. Which Windows Settings category should be used?

## Answer

Personalization.

## Instructor Notes

Personalization changes how Windows looks and feels. It does not manage user
identity or sign-in methods; those belong under Accounts.
