## Question

A user wants the modern Windows Settings category for uninstalling an
installed application. Which category should be opened?

## Answer

Apps.

## Instructor Notes

Apps manages installed applications and optional features in Windows
Settings. Programs and Features is the related legacy Control Panel
destination.
