## Question

A user needs to change display scaling, Night light, and notification behavior
in Windows Settings. Which category contains these options?

## Answer

System.

## Instructor Notes

The System category covers display, audio, and notification settings. The
legacy System Control Panel applet instead emphasizes computer information and
paths to remote and protection settings.
