## Question

Two files have similar names and icons, so a technician needs to see their
actual filename suffixes. Which File Explorer setting should be changed?

## Answer

Disable the option that hides extensions for known file types.

## Instructor Notes

Showing extensions exposes the suffix used to identify the file type. Showing
hidden files solves a different visibility problem.
