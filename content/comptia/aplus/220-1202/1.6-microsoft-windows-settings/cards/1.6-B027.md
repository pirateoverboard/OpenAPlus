## Question

A user wants the Windows Settings category for Bluetooth devices, printers,
mouse behavior, and pen input. Which category should be opened?

## Answer

Devices.

## Instructor Notes

Devices groups peripheral and input configuration in Windows Settings. Device
Manager remains the better tool for hardware status and drivers.
