## Question

A support task requires changing the local Windows host-firewall
configuration. Which Control Panel destination should be used?

## Answer

Windows Defender Firewall.

## Instructor Notes

Windows Defender Firewall is the operating system's local firewall interface.
It is distinct from Network and Sharing Center, which manages network and
sharing settings.
