## Question

A laptop must preserve open work on disk before powering down so the session
does not depend on continuous memory power. Which power state should be used?

## Answer

Hibernate.

## Instructor Notes

Hibernate saves the session to disk. Sleep keeps the active session in memory
for a quicker return but continues to require some power.
