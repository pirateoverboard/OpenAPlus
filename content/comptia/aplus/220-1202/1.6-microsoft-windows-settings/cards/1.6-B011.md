## Question

Windows searches need to include an additional folder that is not currently
indexed. Which Control Panel destination should the technician modify?

## Answer

Indexing Options.

## Instructor Notes

Indexing Options controls which locations Windows indexes to speed searches.
File Explorer search options influence search behavior but do not replace the
index-location configuration.
