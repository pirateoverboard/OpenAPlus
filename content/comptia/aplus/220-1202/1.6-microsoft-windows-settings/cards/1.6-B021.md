## Question

A user needs to add a Windows display language and correct automatic clock
configuration. Which Windows Settings category contains these options?

## Answer

Time and Language.

## Instructor Notes

Time and Language groups clock, regional, and language configuration. Correct
time is also important for environments that rely on synchronized
authentication.
