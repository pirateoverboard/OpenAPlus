## Question

A user wants a reduced-power state that keeps open applications in memory for
a quick return. Which power state fits the goal?

## Answer

Sleep, also called suspend or standby.

## Instructor Notes

Sleep retains the working session in memory. Hibernate writes the session to
disk and is the better choice when maintaining memory power is undesirable.
