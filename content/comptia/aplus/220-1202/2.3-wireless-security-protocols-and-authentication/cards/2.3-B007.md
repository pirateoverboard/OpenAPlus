## Question

An organization wants each wireless user authenticated individually by a
central server instead of giving everyone the same pre-shared key. Should it
use WPA Personal or WPA Enterprise?

## Answer

WPA Enterprise with 802.1X and an authentication server such as RADIUS.

## Instructor Notes

Personal mode uses a shared network key. Enterprise mode delegates individual
authentication to a central service while the WPA protocol protects the
wireless traffic.
