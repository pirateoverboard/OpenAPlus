## Question

A public wireless network does not require an authentication password but
should still encrypt each user's wireless traffic. Which Objective 2.3
protocol adds that capability?

## Answer

WPA3.

## Instructor Notes

WPA3 can protect traffic on an open network. “Open” describes the lack of a
shared authentication password; it does not require the radio traffic to be
sent without encryption.
