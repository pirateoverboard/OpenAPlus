## Question

A technician is entering a static IPv4 configuration. Which value tells the
client which destination addresses are on its local subnet?

## Answer

The subnet mask.

## Instructor Notes

The mask defines the local subnet boundary. The default gateway has a different
role: it is used when the destination lies outside that local subnet.
