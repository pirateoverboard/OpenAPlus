## Question

A Windows client can communicate with hosts on its local subnet but cannot
reach any off-subnet IP address. Which client setting should be checked?

## Answer

The default gateway.

## Instructor Notes

The default gateway routes traffic beyond the local subnet. Because the
scenario already confirms local IP communication, the subnet's immediate link
is working and the gateway setting is the relevant client value.
