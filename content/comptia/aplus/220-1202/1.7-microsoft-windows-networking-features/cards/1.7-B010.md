## Question

A Windows client must always use specific administrator-provided IP settings
instead of requesting them automatically. Which addressing method should be
configured?

## Answer

Static addressing.

## Instructor Notes

Static configuration manually fixes the client's IP parameters. The
technician must supply correct values for the local network rather than rely on
DHCP.
