## Question

A remote employee needs to use the built-in Windows client to connect to the
workplace network with assigned credentials. Which network connection should
the technician configure?

## Answer

A VPN connection.

## Instructor Notes

Windows includes a VPN client for workplace connections. Depending on the
organization, authentication may use entered credentials or a smart card.
