## Question

A Windows client is using a usage-billed link. Which connection setting tells
Windows and supported applications to reduce some background data use?

## Answer

Mark the connection as metered.

## Instructor Notes

Metering is useful for capped, costly, or slow links and can reduce activities
such as updates or synchronization. It does not correct invalid IP, DNS, or
gateway settings.
