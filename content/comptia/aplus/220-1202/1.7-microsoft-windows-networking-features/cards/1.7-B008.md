## Question

No predefined firewall exception matches a required inbound port. Which type
of Windows Defender Firewall rule gives the technician control over that port?

## Answer

A custom rule.

## Instructor Notes

Predefined exceptions cover common cases. A custom rule is appropriate when
the technician must explicitly allow or block a port or apply other detailed
conditions.
