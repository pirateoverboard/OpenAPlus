## Question

A user repeatedly accesses a file-server share and wants it to appear in File
Explorer with a local drive letter. What should the technician configure?

## Answer

A mapped network drive.

## Instructor Notes

Mapping associates a drive letter with a server share and can reconnect it
automatically. Entering the network path directly reaches the same resource
without creating the drive-letter association.
