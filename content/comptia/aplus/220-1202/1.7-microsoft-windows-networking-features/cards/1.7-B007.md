## Question

A Windows client already has allowed-app exceptions, but the current network
requires the most restrictive inbound setting. Which firewall option ignores
the exception list?

## Answer

Block all incoming connections.

## Instructor Notes

This setting overrides ordinary allowed-app exceptions for the applicable
network profile. It is a deliberate restrictive mode, not the normal way to
permit a trusted application.
