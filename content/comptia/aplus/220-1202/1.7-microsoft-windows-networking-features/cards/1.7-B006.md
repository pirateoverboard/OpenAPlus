## Question

An approved application needs inbound network access, but the Windows firewall
must remain enabled. What should the technician configure?

## Answer

An application or feature exception in Windows Defender Firewall.

## Instructor Notes

A targeted exception permits the approved application without removing host
firewall protection for unrelated traffic. Disabling the entire firewall is a
broader and less secure change.
