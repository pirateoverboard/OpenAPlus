## Question

A Windows client should receive its IP configuration automatically from the
network. Should the adapter use static addressing or DHCP?

## Answer

DHCP.

## Instructor Notes

DHCP supplies dynamic client configuration automatically. Static configuration
instead requires the technician to enter the necessary IP parameters.
