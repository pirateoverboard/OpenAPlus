## Question

A mobile Windows client normally uses DHCP, but it needs known static settings
when connected to a network where DHCP is unavailable. What adapter feature
should the technician configure?

## Answer

An alternate static configuration.

## Instructor Notes

Alternate Configuration supplies defined fallback values when DHCP cannot be
reached. Without a configured static fallback, Windows normally uses APIPA.
