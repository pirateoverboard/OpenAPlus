## Question

An organization requires supported client traffic to pass through an
intermediary address, except for specified internal destinations. Which Windows
network setting should be configured?

## Answer

Proxy settings, including the proxy address and bypass exceptions.

## Instructor Notes

A proxy changes the path used by supported application traffic. Exceptions
identify destinations that should bypass it, and not every application or
protocol necessarily uses the same proxy configuration.
