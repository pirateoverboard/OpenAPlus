## Question

A Windows client reaches a server by IP address but not by hostname. Which
client network setting should the technician check?

## Answer

The DNS settings.

## Instructor Notes

Domain Name System settings identify the resolver used to translate names into
IP addresses. Successful access by IP narrows this scenario to name resolution
rather than general routing.
