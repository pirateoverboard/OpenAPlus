## Question

A printer is connected to one Windows PC and must be made available to other
network users. Which property should the technician configure on the host PC?

## Answer

The printer's sharing property.

## Instructor Notes

Windows shares an existing printer through Printer Properties. Other users can
then add the shared printer through File Explorer or Windows Settings.
