## Question

A laptop must connect through a mobile carrier when neither Ethernet nor Wi-Fi
is the intended access method. Which Windows network connection type applies?

## Answer

WWAN, or cellular networking.

## Instructor Notes

Wireless wide area networking uses compatible cellular hardware and carrier
service. Provider-specific software and setup can vary, so confirm the adapter
and service requirements.
