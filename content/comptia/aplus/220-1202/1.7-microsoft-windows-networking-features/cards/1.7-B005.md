## Question

A technician wants to open the `Projects` share on a server named `File01`
directly in File Explorer. What network path should be entered?

## Answer

`\\File01\Projects`

## Instructor Notes

A Windows network path identifies the server first and the share second. The
same path can later be assigned a drive letter if persistent mapped access is
useful.
