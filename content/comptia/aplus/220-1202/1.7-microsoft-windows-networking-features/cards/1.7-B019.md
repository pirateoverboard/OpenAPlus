## Question

A user connects a Windows laptop to untrusted café Wi-Fi and does not need
local device sharing. Should the technician select the public or private
network profile?

## Answer

The public network profile.

## Instructor Notes

Public is the restrictive choice for an untrusted network. Private is intended
for a trusted home or work network where local discovery and sharing are
needed; firewall behavior can differ by profile.
