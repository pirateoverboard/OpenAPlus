## 2026-07-12 - Objective 1.4 draft authoring

- Added the Core 2 Objective 1.4 study guide for Task Manager, MMC snap-ins,
  and additional Windows tools.
- Added 20 Basic cards and 14 Command cards after
  independent-review fixes.
- Used CompTIA A+ 220-1202 Exam Objectives v4.0 Objective 1.4 as the primary
  scope authority.
- Used Professor Messer 220-1202 v1.40 pages 7-9 as the approved secondary
  validation/page-reference source.
- Completed a private Professor Messer Core 2 practice-exam gap check; it
  reinforced Event Viewer, Task Manager, Task Scheduler, Disk Management, and
  Group Policy tool-selection coverage without being cited in card or
  study-guide source fields.
- Image cards were considered and omitted because this objective tests tool
  selection and command/tool recall rather than visual recognition.

### Scope and source decisions

- Practice-exam content was used only as a private gap check and did not define
  scope or supply card wording.
- Tool command names were carded as typed Command facts because Objective 1.4
  explicitly lists the tool executables and `.msc` launch names.
- Tool behavior was carded separately from command recall to avoid duplicate
  Basic/Command learning targets.
- No new secondary sources were used.
