## Question

Which Windows MMC snap-in should you use to manage local user accounts and
local group membership?

## Answer

Local Users and Groups.

## Instructor Notes

Use Local Users and Groups for local accounts and group membership. Domain user
management belongs to directory services tools outside this objective's local
tool focus.
