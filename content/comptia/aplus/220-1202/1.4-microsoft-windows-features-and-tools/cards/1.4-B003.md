## Question

A user complains that many applications open automatically after sign-in. Which
Task Manager area should you use to disable unnecessary sign-in startup apps?

## Answer

The Startup tab.

## Instructor Notes

Startup focuses on programs that launch when the user logs in. Processes shows
what is running now, but Startup controls future sign-in behavior.
