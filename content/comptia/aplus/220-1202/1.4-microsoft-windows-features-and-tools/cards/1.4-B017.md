## Question

A Windows computer is low on space and you want a built-in tool to remove
unused or unnecessary files. Which tool should you use?

## Answer

Disk Cleanup.

## Instructor Notes

Disk Cleanup is for freeing space by selecting categories of unneeded files. It
does not reorganize file fragments or manage partitions.
