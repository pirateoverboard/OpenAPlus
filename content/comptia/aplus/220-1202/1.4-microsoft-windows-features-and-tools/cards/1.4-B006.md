## Question

A technician needs to review yesterday's application crash and related Windows
log entries. Which MMC snap-in should they use?

## Answer

Event Viewer.

## Instructor Notes

Event Viewer is for consolidated historical logs such as application, security,
setup, and system events. It is not the best tool for live performance
measurement.
