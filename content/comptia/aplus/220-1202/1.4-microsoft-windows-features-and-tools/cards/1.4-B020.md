## Question

What is the purpose of the Microsoft Management Console in Windows
administration?

## Answer

It provides a framework for loading administrative snap-ins such as Event
Viewer, Disk Management, Task Scheduler, and Local Users and Groups.

## Instructor Notes

The console itself is a container. In support scenarios, the specific snap-in is
usually the important tool selection.
