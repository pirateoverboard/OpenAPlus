## Question

Task Manager shows high disk activity, but you need a more detailed live view
by CPU, memory, disk, and network categories. Which tool should you open?

## Answer

Resource Monitor.

## Instructor Notes

Resource Monitor provides a deeper real-time breakdown than Task Manager. It is
not the same as Performance Monitor, which is better for longer-term collection
and reports.
