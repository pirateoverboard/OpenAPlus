## Question

Which Windows tool is used to view and manage user and trusted certificates?

## Answer

Certificate Manager.

## Instructor Notes

Certificate Manager is the objective-listed tool for certificate stores. Do not
confuse it with Device Manager, which manages hardware devices and drivers.
