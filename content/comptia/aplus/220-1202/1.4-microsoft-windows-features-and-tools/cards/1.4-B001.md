## Question

A Windows user says the computer is slow right now. Which built-in tool should
you open first to quickly view current CPU, memory, disk, and network activity?

## Answer

Task Manager.

## Instructor Notes

Task Manager is the fast local view for current resource pressure. Event Viewer
is better for historical logs, and Performance Monitor is better for longer-term
collection.
