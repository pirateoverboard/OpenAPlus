## Question

A support procedure requires changing a Windows Registry value. Which tool is
used, and what precaution should you take first?

## Answer

Use Registry Editor and back up the registry or the relevant keys before
editing.

## Instructor Notes

The registry is a central configuration database. Registry edits can affect
Windows, services, drivers, applications, and user interface behavior.
