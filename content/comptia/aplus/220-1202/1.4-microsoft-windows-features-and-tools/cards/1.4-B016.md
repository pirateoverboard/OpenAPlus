## Question

Which Windows tool centralizes boot, startup, and service configuration for
controlled startup troubleshooting?

## Answer

System Configuration.

## Instructor Notes

System Configuration can change how Windows boots and which startup items or
services are active. That makes it useful but also potentially disruptive.
