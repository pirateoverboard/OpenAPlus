## Question

A maintenance script needs to run automatically every Friday evening. Which
Windows tool should you use?

## Answer

Task Scheduler.

## Instructor Notes

Task Scheduler is the built-in tool for scheduled or triggered applications and
scripts. Task Manager shows running work; it does not create recurring jobs.
