## Question

Which Windows tool should you use to view a graphical disk and volume layout
before creating, deleting, or changing partitions?

## Answer

Disk Management.

## Instructor Notes

Disk Management can change disk structures and erase data. Verify the selected
disk and backup status before making destructive storage changes.
