## Question

Which Windows tool is the better choice for collecting long-term CPU, memory,
and disk statistics with alerts and reports: Task Manager or Performance
Monitor?

## Answer

Performance Monitor.

## Instructor Notes

Task Manager is quick and immediate. Performance Monitor is built for longer
collection, stored statistics, alerts, automated actions, and reports.
