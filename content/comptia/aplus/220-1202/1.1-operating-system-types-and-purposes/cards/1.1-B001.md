## Question

What is the primary purpose of an operating system?

## Answer

An operating system coordinates hardware resources, applications, and user
interaction so the computer can run usable workloads.

## Instructor Notes

The exam is not asking for one narrow feature such as a desktop or file
manager. The OS is the control layer between hardware, software, and the user.
