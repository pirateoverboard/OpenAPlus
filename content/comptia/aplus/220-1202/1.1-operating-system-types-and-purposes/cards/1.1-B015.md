## Question

Why does moving a native application from one operating system to another often
fail even when the user files can be copied?

## Answer

Native applications are usually built for a specific OS platform, while many
data files or web-based services are more portable across systems.

## Instructor Notes

The exam distinction is application compatibility versus data portability. A
file may move successfully even when the application that created it cannot run
on the destination OS.
