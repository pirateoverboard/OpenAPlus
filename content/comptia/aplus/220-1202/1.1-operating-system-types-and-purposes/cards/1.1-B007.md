## Question

Why must a partition be formatted with a filesystem before it stores normal
files?

## Answer

Formatting creates the filesystem structure the operating system uses to
organize files and metadata on that partition.

## Instructor Notes

The filesystem is not just a label. It defines how data is arranged and which
OS features and compatibility limits apply to the storage device.
