# Cloze guide

Cloze cards test precise, compact recall. Use them for definitions, acronyms,
ports, commands, protocols, standards, short facts, and other memorization where
the surrounding sentence provides useful context.

## Construction rules

- Delete the meaningful answer, not filler words such as “is,” “the,” or “and.”
- Keep the sentence understandable before and after the deletion.
- Normally test one fact per note. One or two cloze numbers are usually enough.
- Use the same cloze number only when the hidden items form one answer.
- Use different numbers when each deletion should create a separate review.
- Keep the answer short. Move explanation and contrast into `## Extra` or
  `## Instructor Notes`.

**Good:** “`[acronym]` expands to {{c1::[full name]}}.”

**Good:** “The command for `[single operation]` is {{c1::[command]}}.”

**Bad:** “The {{c1::system}} {{c2::is}} {{c3::used}} for
{{c4::many unrelated purposes}}.” Most deletions are filler or overly broad.

**Bad:** A long paragraph in which nearly every noun is hidden. This rewards
sentence memorization and creates excessive sibling cards.

## When to use Basic instead

Convert a Cloze card to a Basic scenario card when the learner must:

- interpret symptoms or constraints;
- choose the MOST likely cause;
- select the BEST next step or FIRST check;
- compare alternatives or explain why;
- apply a fact rather than reproduce it.

If meaningful context makes the answer obvious without understanding, Cloze is
the wrong format.
